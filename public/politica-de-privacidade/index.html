Política para Clientes, Prestadores e Parceiros Digitais

A Plataforma Nobis se empenha a manter a transparência necessária no manejo dos dados pessoais de seus colaboradores, clientes, usuários, empresas associadas e candidatos ao programa de parceria digital. Assim, se compromete a tratar os dados pessoais em conformidade com os princípios de proteção de dados prescritos na Lei n. 13.709/18 - Lei Geral de Proteção de Dados (“LGPD”) e outras leis relevantes.


1. ESG.NOBISAPP.COM.BR

1.1. Quais dados pessoais são coletados?

Nome
CPF/CNPJ
Celular
E-mail
Dados do cartão de crédito
Endereço de cobrança

1.2. Como estes dados pessoais são protegidos?

Estas informações não são salvas em nosso banco de dados, elas são utilizadas apenas para realizar a transação na área de projetos.

1.3. Para que servem estes dados pessoais?

Os dados mencionados não são salvos pela Nobis. No website, ao realizar uma doação, pedimos para que o usuário insira os dados de cobrança apenas para realizar esta transação. Desta forma, uma vez que a transação é finalizada através da nossa API, os dados não ficam salvos em nosso banco de dados.

1.4. Como compartilhamos seus dados pessoais?

Novamente, os dados não são compartilhados e são visualizados internamente apenas por funcionários relacionados ao gerenciamento dos projetos.


2. OBSERVATORIO.NOBISAPP.COM.BR

2.1. Quais dados pessoais são coletados?

Nome
E-mail
Telefone
CPF
Sexo
Data de nascimento
Profissão
Empresa empregadora
Área de atuação
Dados do cartão

2.2. Como estes dados pessoais são protegidos?

Todas as senhas cadastradas pelos usuários são salvas em nosso banco apenas após passarem por dois processos distintos de criptografia.

2.3. Para que servem estes dados pessoais?

Estes dados servem para gerar estatísticas internamente para que seja possível a gente saber para quem nossos serviços são relevantes.

2.4. Como compartilhamos seus dados pessoais?

Os dados coletados através do cadastro do Observatório Nobis não são compartilhados, e são vistos apenas internamente. Além disso, todo e qualquer dado de cartão de crédito não é salvo em nosso banco de dados interno, é usado apenas para realizar a transação da mensalidade.


3. PARCEIROS.NOBISAPP.COM.BR

3.1. Quais dados pessoais são coletados?

Nome
E-mail
Data de nascimento
Endereço

3.2. Como estes dados pessoais são protegidos?

Todas as senhas cadastradas pelos usuários são salvas em nosso banco apenas após passarem por dois processos distintos de criptografia.

3.3. Para que servem estes dados pessoais?

No caso do website parceiros.nobisapp.com.br os dados coletados servem para controle interno na empresa dos possíveis candidatos a parcerias.

3.4. Como compartilhamos seus dados pessoais?

Estes dados não são compartilhados. Eles são visualizados internamente e apenas pelos funcionários que tem relação com os parceiros afiliados.


4. INSTITUTO.NOBISAPP.COM.BR

4.1. Quais dados pessoais são coletados?

Nome
E-mail
Linkedin
CPF/CNPJ *
Celular *
Dados do cartão de crédito *
Endereço de cobrança *

4.2. Como estes dados pessoais são protegidos?

Estas informações não são salvas em nosso banco de dados, elas chegam em um e-mail para o responsável da área do Instituto.

4.3. Para que servem estes dados pessoais?

Os dados marcados com * não são salvos pela Nobis. No Instituto, ao realizar uma doação, pedimos para que o usuário insira os dados de cobrança apenas para realizar esta transação. Desta forma, uma vez que a transação é finalizada através da nossa API, os dados não ficam salvos em nosso banco de dados. As informações coletadas pelo site do Instituto, entretanto, servem para que a gente do Instituto Nobis possa entrar em contato com os responsáveis de projetos que estejam interessados em fazer uma parceria e vincular seus projetos com o Instituto Nobis.

4.4. Como compartilhamos seus dados pessoais?

Novamente, os dados não são compartilhados e são visualizados internamente apenas por funcionários relacionados ao Instituto Nobis.
